## Multi-Sensor Visualisation Analysis
# Project Overview
This project simulates and visualises 48 hours of industrial sensor data to identify correlations and operational patterns. By generating realistic streams for Temperature, Vibration, Current, and Humidity, the tool demonstrates how data science can be applied to machine health monitoring and predictive maintenance.

# Key Features
Realistic Data Simulation: Models diurnal cycles and cross-sensor correlations (e.g., Vibration linked to Motor Current).

Multi-Panel Visualisation: A 4-panel time series figure for temporal analysis.

Statistical Analysis: Distribution plots and a correlation heatmap to identify hidden relationships.

Automated Reporting: Generates a self-contained HTML report with embedded visualisations.

# Visualisation Components
The project generates three primary analysis views:

- Time Series: Tracks sensor fluctuations over a 48-hour window.

- Distribution Panel: Displays the density and spread of sensor values using KDE plots.

- Correlation Heatmap: Quantifies the strength of relationships between different sensor metrics.
