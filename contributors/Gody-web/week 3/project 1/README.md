Vector Operations VisualiserWeek 3: Mathematical Foundations for MLOverviewThis project visualizes how Vector Geometry is used to monitor industrial sensors. It demonstrates how a machine "decides" if a reading is normal or anomalous using a mathematical boundary.Key Features2D/3D Visualization: Plots Temperature, Current, and Voltage as vectors in feature space.Vector Operations: Implements Dot Product and Cosine Similarity from scratch using NumPy.Sensor Journey: An animation of 50 live sensor readings.Decision Boundary: A weight-vector-based "fence" that separates normal data (green) from anomalies (red).Math ImplementationThe project relies on the Dot Product to classify data:Formula: $\mathbf{w} \cdot \mathbf{x} = \sum w_i x_i$Logic: If the Dot Product is $\geq 0$, the system is Normal; if $< 0$, it is an Anomaly.Requirementsnumpy (Linear Algebra)matplotlib (Animation & 3D Plotting)
