# Project 2: Electrical Calculations Toolkit
This is a command-line tool using `input()` for user interaction that computes Ohm’s Law, Power, RC time constants, and series/parallel resistance. Each formula is its own function. User selects which calculation to perform from a menu.

## Features
- Interactive command-line menu using `input()`
- Calculates Ohm’s Law: $V = IR$
- Calculates Power: $P = VI$, $P = I^2R$, $P = V^2/R$
- Calculates RC time constant: $\tau = RC$
- Calculates total resistance for series: $R_{total} = R_1 + R_2 + … + R_n$
- Calculates total resistance for parallel: $\frac{1}{R_{total}} = \frac{1}{R_1} + \frac{1}{R_2} + … + \frac{1}{R_n}$
- Each formula implemented as its own function
- Menu-driven selection of calculations
