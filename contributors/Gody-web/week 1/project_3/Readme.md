# Project 3: Data Logger Simulator

This project is to simulate a 24-hour sensor log with readings every 5 minutes (288 records) and write to CSV. Read it back and compute hourly averages using only built-in Python — no NumPy yet. Print a table of hourly stats.

## Features
- Simulates 24 hours of sensor data at 5-minute intervals
- Generates 288 total records
- Writes all readings to CSV
- Reads CSV back using built-in file I/O
- Computes hourly averages with standard Python only
- Prints formatted table of hourly statistics

## Files
| File | Description |
	
| `data_logger.py` | Main script to simulate, log, and analyze sensor data |
| `sensor_log.csv` | Output CSV containing 288 readings over 24 hours |
